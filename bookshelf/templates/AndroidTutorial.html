Hi, Welcome to Our Android App Tutorial!

Today you'll be learning our app's basic functionalities, and how to use them!

Note: The app is intended for user's use only, and admin abilities are only available in the online web browser edition.

Launch Page:

This is what you will see upon launching the app. You have two options:

  1. Sign Up
  2. Login

Let's Try Signing up first

Signup Page:

From here you can either traverse back to the login menu if you already have an account, or you can create an account. Upon creation, you will be taken to the login page once more to login. Let's take a look at happens when you log in successfully.

Main Menu:

Take a look at the main menu:

You have eight options for you to choose from:

  1. Create an Event
  2. Join an Event
  3. See all Events
  4. See my Events
  5. Search Events
  6. Search for Open Venues on a Date/Time
  7. Search for Timeslots Open at a Venue on a Date
  8. Logout

Create an Event:

In the event add page, there are 9 text fields: Event Name, Host Name, Description, TimeSlot(8-12), Current Users, Max Users, Price and venue ID. After all fields are filled, the user creates the event by clicking The “Save” button.

Join an Event

Here a User can join an event by either putting the event name or the event ID in the fields. If the user does not know either, it is recommended that the user preforms a search to find the values.

See all Events

On this page the user can look through all events in the system

See My Events

On this page a user can see all the events they have signed up for.

Search Events

On this page a user can filter through events using: eventname, date & time, and Venue ID.

Search for Open Venues on a Date time

Here just specify a date and Venue ID, and the app will tell you all the timeslots available at a particular venue on that day.

Search for Open Timeslots at a Venue on a day.

Here, simply enter in a date and timeslot, and the app will tell you the venues that are available to book at that time.

Logout

Finally, here is the logout page. A user can confirm their logout attempt, or can travel back to the main menu.
